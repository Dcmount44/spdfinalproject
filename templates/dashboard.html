User Dashboard (dashboard.html) Purpose: A personal space for the user to manage their listings, messages, and reviews. Features: Overview of the user’s resources (list of items they’ve listed with options to edit or delete). Display messages between users related to resource reservations or other inquiries. Reviews: User ratings and feedback received from others, and the option to leave reviews for others. Link to the profile page for editing profile details. Navigation menu for adding new resources, viewing reservations, etc.
